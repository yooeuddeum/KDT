JavaScript

숫자 Number(Value)란?

Number()는 Number을 생성한다는 의미이고, Number(Value)의 의미는 만들어 낼 수 있는 숫자의 의미입니다
큰 틀로 봤을 때 정수(integer)와 실수(float)가 있습니다 정수는 분수나 소수점이 없는 수 입니다
예시로는 0, 1, 2, 3이 되겠습니다 반대로 실수는 소수점이 있는 수 입니다 예시로는 0.1, 0.2, 0.3이 되겠습니다

문자(charactor)란?

문자란 문자 값 앞뒤를 단일 인용 부호 ''로 감싸 주어야 합니다
문자의 예시로는 이름으로 기준으로 뒀을때 "유으뜸"에 있어서
"유" 부분이 되겠습니다 그 외에 특수문자도 해당이 될 것 같습니다

문자열(String)이란?

문자열이란 기호의 순차 수열이며, 문자들이 모여서 만들어지는 집합체 입니다
또한 텍스트 형식으로 표현 가능한 데이터를 저장하는데 유용합니다 그리고
single, double, quote 변수와 구분 짓기 위해 따옴표를 쓰기도 하고, 계산 할 수 없는 텍스트 입니다.

Boolean, bool이란?

Boolean, bool은 크게 봐서 두가지의 정의가 있습니다 참(true),거짓(false)가 있습니다 이로 인해서 사람들이 수동적으로 핸들링이 가능한 부분이며 참과 거짓에 의미가 맞다와 아니다의 차이일 뿐 잘했다와 못했다의 개념이랑은 다릅니다 또한 문자열이 아니므로 따옴표로 감싸지 않는다는 특징이 있습니다
예시로는 내가 20살보다 많으면 참으로 해주고 20살 보다 어리면 거짓으로 해줘 같은 개념이 되겠습니다

묶음, Object(오브젝트), 객체란?

객체는 데이터와 함수 집합 입니다 하위 그룹을 묶어 놓은 상징성이 있고
부모요소의 개념으로 알고있다 그리고 여러 개의 속성과 값을 담을 수 있다는 점이 좀 다르다 속성과 값은 클론으로 구분을 하며 함수를 담을 수 있습니다. 이러한 경우에는 메소드를 생성하는 결과가 됩니다. 예시로 가방이 객체가 될 것이고 가방안에
있을 수 있는 필기도구,지갑,핸드폰등등의 묶음들이(property) 될 것 같습니다

배열 , Array , list이란?

배열(Array)은 tostring(object를 표현하는 문자열)과 같은 객체 속성입니다
사용하는 데이터 타입은"[ ]" 대괄호를 사용하며 대괄호안에는 인덱스(자료구조)를
사용하고 전 인덱스를 각각 하나의 덩어리들이라고 생각합니다
또한 정수 및 해당 수의 문자열 형식으로만 사용할 수 있습니다 순서로는 배열의 첫번째 인덱스는 0번부터 시작하게 되며, 배열은 항목 이름이 굳이 필요 없는 데이터 일 때 편하게 사용하려고 만든 객체 타입이기도 합니다.

함수(function)란?

함수는 하나의 상자라는 개념으로 생각하고 있고, 내가 만들고 싶은 것을 가능하게 해주는 기능으로 생각하고 있습니다
또한 임의로 내가 함수를 만들어 놓고 수동적으로 필요로 할 때 사용할 수 있고, 전반적인 개념은 선언을 하고 호출을 한다라는 개념이 될 것 같습니다
함수는 내가 만든 함수가 있고 남이 만든 함수로 예를 두게 된다면 내가 만든 함수는 나만의 아이디어로 만들게 된 코드를 나만의 방식으로 만든 함수를 내가 만든 함수의 개념으로
이해하면 될것 같고 남이 만든 함수는 개발자들 사이에서 룰 같은 개념으로 편리성을 위해 만들어진걸 공유하면서 사용하게 되는걸 남이 만든 함수로 생각하면 될 것 같습니다(공식 홈페이지)
함수는 입력,출력,실행 이 3가지의 키워드로 기억하면 될 것 같습니다 웹에 관련된건 대부분 실행을 시킨다(반복=자동화)의 개념이 될 것 같습니다.
